# CareerPilot AI

## Project Overview

CareerPilot AI is an Android application that helps students and freshers create professional resumes and cover letters using AI.

The application should be modern, fast, simple, and optimized for mobile devices.

---

# MVP Features

- User Authentication
- Dashboard
- Resume Builder
- Resume Preview
- PDF Export
- AI Professional Summary
- AI Cover Letter Generator

---

# Future Features
